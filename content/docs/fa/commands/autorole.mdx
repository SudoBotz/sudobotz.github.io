## سیستم اتو تگ چیست؟

<Callout type="info" title="توضیحات سیستم">
سیستم اتو تگ به شما این امکان را می‌دهد که وقتی یک رول خاص به کاربری داده می‌شود، به صورت خودکار یک تگ (برچسب) به ابتدا یا انتهای نام آن کاربر اضافه شود.

**مثال**: اگر رول `Developer` با تگ `Dev` تنظیم کرده باشید:
- قبل از دادن رول: `AmirKenzoo`
- بعد از دادن رول: `Dev AmirKenzoo` یا `AmirKenzoo Dev`
</Callout>

## ساخت تگ برای رول

### تگ در ابتدای نام

<Callout type="info" title="افزودن تگ به ابتدای نام">
برای اضافه کردن تگ به ابتدای نام کاربر:
```
/stp autorole tag role:@【 𝐅𝐨𝐮𝐧𝐝𝐞𝐫 】 tag:👑 position:ابتدای نام
```
با دادن رول **【 𝐅𝐨𝐮𝐧𝐝𝐞𝐫 】** به کاربر، تگ `👑` به ابتدای نام او اضافه می‌شود.

**نتیجه**: `👑 Kenzo`
</Callout>

### تگ در انتهای نام

<Callout type="info" title="افزودن تگ به انتهای نام">
برای اضافه کردن تگ به انتهای نام کاربر:
```
/stp autorole tag role:@【 𝐃𝐞𝐯 】 tag:dv position:انتهای نام
```
با دادن رول **【 𝐃𝐞𝐯 】** به کاربر، تگ `dv` به انتهای نام او اضافه می‌شود.

**نتیجه**: `Kenzo dv`
</Callout>

### مثال‌های بیشتر

<Callout type="info" title="نمونه‌های مختلف تگ">
```
/stp autorole tag role:@Staff tag:【STAFF】 position:ابتدای نام
/stp autorole tag role:@VIP tag:⭐ position:ابتدای نام
/stp autorole tag role:@Moderator tag:MOD position:انتهای نام
/stp autorole tag role:@Admin tag:ADM position:ابتدای نام
```

این دستورات تگ‌های مختلفی برای رول‌های مختلف ایجاد می‌کنند:
- **Staff** → `【STAFF】 Kenzo `
- **VIP** → `⭐ Kenzo`
- **Moderator** → `Kenzo MOD`
- **Admin** → `ADM Kenzo`
</Callout>

## نمایش لیست تگ‌ها

<Callout type="info" title="مشاهده تمام تگ‌های ثبت شده">
برای دیدن لیست کامل رول‌ها و تگ‌های تنظیم شده:
```
/stp autorole list
```

**نمونه خروجی**:
```
تگ‌های خودکار
@【 GLɪᴛCH 】 → ᴳᴸᴵᵀᶜᴴ (ابتدای نام)
@【 𝐃𝐞𝐯 】 → dv (انتهای نام)
@Staff → 【STAFF】 (ابتدای نام)
```

این لیست همه رول‌هایی که برایشان تگ تنظیم کرده‌اید را نمایش می‌دهد.
</Callout>

## حذف تگ از رول

<Callout type="info" title="حذف تگ یک رول مشخص">
برای حذف تگ از یک رول خاص:
```
/stp autorole remove role:@【 𝐃𝐞𝐯 】
```
این دستور تگ تنظیم شده برای رول **【 𝐃𝐞𝐯 】** را حذف می‌کند.

⚠️ توجه: پس از حذف، تگ دیگر برای کاربران جدیدی که این رول را دریافت می‌کنند اعمال نمی‌شود.
</Callout>

### مثال‌های حذف

<Callout type="info" title="نمونه‌های حذف تگ">
```
/stp autorole remove role:@Staff
/stp autorole remove role:@VIP
/stp autorole remove role:@Moderator
```
هر کدام از این دستورات، تگ رول مربوطه را حذف می‌کنند.
</Callout>

## پاک کردن همه تگ‌ها

<Callout type="warn" title="حذف کامل تمام تگ‌ها">
برای پاک کردن تمام تگ‌های تنظیم شده:
```
/stp autorole clear
```

⚠️ **هشدار**: این دستور تمام تگ‌های تنظیم شده برای همه رول‌ها را به صورت دائمی حذف می‌کند!

این عملیات قابل بازگشت نیست، پس قبل از اجرا مطمئن شوید.
</Callout>

## نکات مهم

<Callout type="warn" title="نکات استفاده">
1. **انتخاب رول**: حتماً رول را با `@` منشن کنید یا از منوی انتخاب استفاده کنید
2. **طول تگ**: تگ‌های خیلی طولانی ممکن است نام کاربر را نامناسب کنند
3. **کاراکترهای خاص**: می‌توانید از ایموجی و کاراکترهای یونیکد استفاده کنید
4. **موقعیت**: فقط دو گزینه موجود است: **ابتدای نام** یا **انتهای نام**
5. **حذف خودکار**: با حذف رول از کاربر، تگ به صورت خودکار از نام حذف می‌شود
</Callout>

<Callout type="info" title="توصیه‌های طراحی تگ">
برای طراحی تگ‌های زیبا و حرفه‌ای:

✅ **توصیه می‌شود**:
- از تگ‌های کوتاه و خوانا استفاده کنید
- از کاراکترهای خاص برای زیبایی بیشتر استفاده کنید
- تگ‌ها را متناسب با اهمیت رول انتخاب کنید
- از ایموجی‌های مناسب استفاده کنید (مثل ⭐ برای VIP)

❌ **توصیه نمی‌شود**:
- تگ‌های خیلی طولانی (بیش از 10 کاراکتر)
- استفاده از کاراکترهای غیرقابل خواندن
- ایجاد تگ‌های مشابه برای رول‌های مختلف
- استفاده بیش از حد از ایموجی
</Callout>

## مثال‌های کاربردی

<Callout type="info" title="پکیج تگ برای یک سرور گیمینگ">
```
/stp autorole tag role:@Owner tag:👑 position:ابتدای نام
/stp autorole tag role:@Admin tag:⚡ position:ابتدای نام
/stp autorole tag role:@Moderator tag:🛡️ position:ابتدای نام
/stp autorole tag role:@VIP tag:⭐ position:ابتدای نام
/stp autorole tag role:@Pro Player tag:🎮 position:انتهای نام
```

**نتایج**:
- Owner → `👑 Kenzo`
- Admin → `⚡ Kenzo`
- Moderator → `🛡️ Kenzo`
- VIP → `⭐ Kenzo`
- Pro Player → `Kenzo 🎮`
</Callout>

<Callout type="info" title="پکیج تگ برای یک سرور RP">
```
/stp autorole tag role:@Founder tag:【FOUNDER】 position:ابتدای نام
/stp autorole tag role:@Developer tag:【DEV】 position:ابتدای نام
/stp autorole tag role:@Police tag:🚔 position:انتهای نام
/stp autorole tag role:@Medic tag:🚑 position:انتهای نام
/stp autorole tag role:@Gang Leader tag:👑 position:انتهای نام
```

**نتایج**:
- Founder → `【FOUNDER】Kenzo`
- Developer → `【DEV】Kenzo`
- Police → `Kenzo 🚔`
- Medic → `Kenzo 🚑`
- Gang Leader → `Kenzo 👑`
</Callout>

## سوالات متداول

<Callout type="info" title="آیا می‌توانم چند تگ برای یک رول تنظیم کنم؟">
خیر، هر رول فقط می‌تواند یک تگ داشته باشد. اگر دستور `tag` را دوباره برای یک رول اجرا کنید، تگ قبلی جایگزین می‌شود.
</Callout>

<Callout type="info" title="اگر کاربر چند رول با تگ داشته باشد چه می‌شود؟">
اگر کاربری چند رول با تگ داشته باشد، تمام تگ‌ها به ترتیب اولویت رول‌ها به نام او اضافه می‌شوند.
</Callout>

<Callout type="info" title="آیا تگ‌ها روی اعضای فعلی هم اعمال می‌شود؟">
خیر, پس از دادن رول مورد نظر تگ به صورت خودکار به اسم کاربر اضافه میشود
</Callout>
