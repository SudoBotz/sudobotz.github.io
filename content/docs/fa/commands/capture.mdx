## راهنمای دستور

برای دریافت راهنمای سریع دستور می‌توانید از دستور زیر استفاده کنید:

```
/stp capture help
```

## زمان ارسال خودکار

<Callout type="warn" title="زمان ارسال ثابت">
⏰ پیام‌های کپچر به صورت خودکار در **روزهای شنبه، دوشنبه و چهارشنبه** ساعت **20:30 به وقت تهران** ارسال می‌شوند.

⚠️ **توجه**: این روزها و زمان ثابت است و قابل تغییر نیست.

⚠️ **توجه**: برای تنظیم author_name حتما باید author_icon هم وارد بشه
</Callout>

## Placeholder ها

در پیام‌های کپچر می‌توانید از متغیرهای زیر استفاده کنید:

<Callout type="info" title="Placeholder های موجود">
این متغیرها به صورت خودکار با اطلاعات سرور جایگزین می‌شوند:

- `{guild}` - نام سرور
- `{guild_icon}` - لینک آیکون سرور
</Callout>

## فیلدهای قابل تنظیم

<Callout type="info" title="تمام گزینه‌های سفارشی‌سازی">
شما می‌توانید موارد زیر را در پیام کپچر تنظیم کنید:

**محتوای اصلی**
- **title**: عنوان امبد
- **text**: متن/توضیح امبد
- **color**: رنگ امبد (فرمت HEX مثل `#ff8800` یا عدد)

**تصاویر**
- **banner**: URL تصویر اصلی (بنر بالای امبد)
- **thumbnail**: URL تصویر کوچک یا استفاده از `{guild_icon}`

**اطلاعات اضافی**
- **footer_text**: متن فوتر (پایین امبد)
- **footer_icon**: آیکون فوتر
- **author_name**: نام نویسنده (بالای امبد)
- **author_icon**: آیکون نویسنده

**تنظیمات عمومی**
- **channel**: کانال ارسال پیام (مثل `#announcements`)
- **delete_after**: حذف خودکار پیام بر حسب ثانیه (اختیاری)
- **preview**: نمایش پیش‌نمایش پیام (True/False)
</Callout>

## تنظیم پیام کپچر

### مثال ساده با عنوان و متن

<Callout type="info" title="پیام کپچر ساده">
تنظیم یک پیام کپچر با عنوان و متن:
```
/stp capture config title:Capture Tonight text:در {guild} — ساعت 21:30 color:#ff8800
```
این دستور یک پیام کپچر با عنوان سفارشی، متن اطلاع‌رسانی و رنگ نارنجی ایجاد می‌کند.
</Callout>

### تنظیم کانال و بنر

<Callout type="info" title="تنظیم کانال و تصویر بنر">
مشخص کردن کانال ارسال و افزودن بنر:
```
/stp capture config channel:#announcements banner:https://.../cap.png
```
پیام در کانال مشخص شده ارسال می‌شود و یک تصویر به عنوان بنر نمایش داده می‌شود.
</Callout>

### استفاده از حذف خودکار

<Callout type="info" title="حذف خودکار پیام">
تنظیم حذف خودکار پیام بعد از مدت زمان مشخص:
```
/stp capture config delete_after:7200
```
پیام بعد از 7200 ثانیه (2 ساعت) به صورت خودکار حذف می‌شود.
</Callout>

### تنظیم حذف خودکار همراه با کانال

<Callout type="info" title="کانال + حذف خودکار">
ترکیب تنظیمات کانال و حذف خودکار:
```
/stp capture config channel:#announcements banner:https://.../cap.png delete_after:7200
```
پیام در کانال اعلانات ارسال و بعد از 2 ساعت حذف می‌شود.
</Callout>

### استفاده از آیکون سرور

<Callout type="info" title="thumbnail با آیکون سرور">
استفاده از آیکون سرور به عنوان تصویر کوچک:
```
/stp capture config thumbnail:{guild_icon} footer_text:کپچر امشب!
```
آیکون سرور به عنوان تصویر کوچک و یک پیام در فوتر نمایش داده می‌شود.
</Callout>

### تنظیم نویسنده و آیکون

<Callout type="info" title="تنظیم بخش نویسنده">
افزودن نام تیم و آیکون سرور:
```
/stp capture config author_name:تیم {guild} author_icon:{guild_icon}
```
نام تیم در بالای امبد با آیکون سرور نمایش داده می‌شود.
</Callout>

### مثال کامل

<Callout type="info" title="پیام کپچر کامل">
تنظیم یک پیام کپچر با تمام امکانات:
```
/stp capture config title:🎮 Capture امشب! text:در سرور **{guild}** — ساعت 21:30 به وقت تهران\n\nآماده باشید! channel:#╭📢】𝐀𝐧𝐧𝐨𝐮𝐧𝐜𝐞𝐦𝐞𝐧𝐭 banner:https://example.com/capture-banner.png color:#ff8800 thumbnail:{guild_icon} footer_text:کپچر خودکار footer_icon:{guild_icon} author_name:سیستم {guild} author_icon:{guild_icon} delete_after:7200 preview:True
```
این دستور یک پیام حرفه‌ای و کامل با تمام المان‌ها ایجاد می‌کند و پیش‌نمایش آن را نمایش می‌دهد.
</Callout>

### استفاده از مقادیر عددی برای تنظیمات سریع

<Callout type="info" title="تنظیم سریع با اعداد">
استفاده از اعداد برای تنظیم سریع (1 = فعال):
```
/stp capture config title:1 text:1 channel:#╭📢】𝐀𝐧𝐧𝐨𝐮𝐧𝐜𝐞𝐦𝐞𝐧𝐭 banner:link color:1 thumbnail:a footer_text:asd footer_icon:asd author_name:asd author_icon:asd delete_after:7000 preview:True
```
این روش برای تنظیم سریع و استفاده از مقادیر پیش‌فرض مناسب است.
</Callout>

## پاک کردن تنظیمات

### ریست کردن همه تنظیمات

<Callout type="warn" title="حذف کامل تنظیمات">
پاک کردن تمام تنظیمات کپچر:
```
/stp capture clear all:True
```
⚠️ این دستور تمام تنظیمات را حذف می‌کند!
</Callout>

### حذف انتخابی تنظیمات

<Callout type="info" title="حذف بخش‌های خاص">
شما می‌توانید فقط بخش‌های خاصی را حذف کنید:
```
/stp capture clear title:True color:True banner:True thumbnail:True footer:True
```
در این مثال، عنوان، رنگ، بنر، تصویر کوچک و فوتر حذف می‌شوند، اما متن و کانال حفظ می‌شوند.
</Callout>

### حذف کامل تمام فیلدها

<Callout type="info" title="حذف تمام فیلدها به صورت جداگانه">
مشخص کردن دقیق تمام فیلدها برای حذف:
```
/stp capture clear all:True title:True text:True color:True banner:True thumbnail:True footer:True author:True channel:True delete_after:True
```
در این حالت تمام تنظیمات به صورت کامل پاک می‌شوند.
</Callout>

<Callout type="info" title="ترکیب حذف و حفظ">
مشخص کردن دقیق چه چیزی حذف و چه چیزی حفظ شود:
```
/stp capture clear all:True title:True text:False color:True banner:True thumbnail:True footer:True author:False channel:False delete_after:True
```
در این حالت:
- ✅ **حذف می‌شود**: title, color, banner, thumbnail, footer, delete_after
- ❌ **حفظ می‌شود**: text, author, channel
</Callout>

## نکات مهم

<Callout type="warn" title="نکات استفاده">
1. **زمان ارسال**: پیام‌ها فقط **شنبه، دوشنبه و چهارشنبه** ساعت 20:30 به وقت تهران ارسال می‌شوند
2. **فرمت رنگ**: رنگ را حتماً با فرمت HEX وارد کنید (مثل `#ff8800`)
3. **لینک تصاویر**: لینک‌های تصاویر باید معتبر و در دسترس باشند
4. **Placeholder ها**: از آکولاد `{}` برای Placeholder ها استفاده کنید
5. **پیش‌نمایش**: قبل از تایید نهایی، از `preview:True` استفاده کنید
6. **کانال**: کانال باید با `#` شروع شود (مثل `#announcements`)
7. **delete_after**: مقدار بر حسب ثانیه است (مثلاً 7200 = 2 ساعت)
</Callout>

<Callout type="info" title="کاربرد">
سیستم کپچر برای ارسال خودکار اطلاع‌رسانی‌های هفتگی (شنبه، دوشنبه، چهارشنبه) در زمان ثابت مناسب است. می‌توانید از آن برای یادآوری رویدادهای منظم، اعلانات کپچر و موارد مشابه استفاده کنید.
</Callout>
