### مثال های استفاده از دستورات
در پایین میتونید نمونه دستورات رو ببینید

### کرفت آیتم

<Callout type="info" title="کرفت آیتم">
جست‌وجو و نمایش مواد لازم برای ساخت آیتم:
```
/craft item:pistol amount:5
```
توی بخش item میتونید اون گان یا اسکین رو سرچ کنید
</Callout>

### ساخت کیف

<Callout type="info" title="ساخت کیف">
انتخاب نوع کیف و تعداد مورد نیاز:
```

/kif kif:KIF100 amount:10

```
</Callout>

### کرفت اسکین

<Callout type="info" title="کرفت اسکین">
انتخاب نوع اسکین و تعداد:
```
/skin skin:GOLD amount:2
```
</Callout>

### اسلحه سطح پایین

<Callout type="info" title="اسلحه سطح پایین">
کرفت اسلحه‌های لول 0 تا 8:
```
/gun1 weapons:HEAVYPISTOL amount:1
```
</Callout>

### اسلحه سطح بالا

<Callout type="info" title="اسلحه سطح بالا">
کرفت اسلحه‌های لول 8 تا 20:
```
/gun2 weapons:CARBINERIFLEMK2 amount:1
```
</Callout>

### کرفت دارو

<Callout type="info" title="کرفت دارو">
انتخاب نوع دارو و تعداد:
```
/daroo daro:ERYTHOPOIETIN amount:10
```
</Callout>

### آیتم گنگ

<Callout type="info" title="آیتم گنگ">
ساخت آیتم‌های مخصوص گنگ:
```
/item item:LAPTOPHACK amount:3
/item item:RADIO amount:1
```
</Callout>

### آهنگری

<Callout type="info" title="آهنگری">
کرفت آیتم‌های آهنگری:
```
/ahangari ahangari:CARBON amount:2
/ahangari ahangari:GunCleaner amount:1
```
</Callout>

### نمایش بر اساس لول

<Callout type="info" title="نمایش بر اساس لول">
نمایش آیتم‌های قابل ساخت در بازهٔ لول مشخص:
```
/craft-level min_level:5 max_level:10
```
اگه دستور بالا رو بزنید ربات ایتم های که از لول 5 تا 10 انلاک میشود رو نمایش میده
</Callout>
