## معرفی سیستم

این سیستم برای خرید و مدیریت مواد اولیه، متریال‌ها و فلزات طراحی شده است. ربات به صورت خودکار پیام‌های کانال‌های لاکر را شنود کرده و متریال‌های گذاشته شده توسط کاربران را حساب می‌کند.

## تنظیم کانال‌های شنود

### افزودن کانال شنود

<Callout type="info" title="تنظیم کانال لاکر">
برای تنظیم کانالی که ربات پیام‌های لاکر را شنود کند:
```
/stp material add channel:#locker1
```

**توضیح:**
- هر زمان کاربری در کانال مشخص شده متریال اضافه کند، ربات آن را حساب می‌کند
- محدودیتی در تعداد کانال‌های شنود وجود ندارد
- می‌توانید چندین کانال مختلف اضافه کنید

**مثال:**
```
/stp material add channel:#locker1
/stp material add channel:#locker2
/stp material add channel:#storage
```
</Callout>

### حذف یک کانال شنود

<Callout type="info" title="حذف کانال خاص">
برای حذف یک کانال خاص از لیست شنود:
```
/stp material remove channel:#locker1
```

این دستور فقط کانال مشخص شده را از لیست حذف می‌کند و سایر کانال‌ها همچنان فعال می‌مانند.
</Callout>

### حذف تمام کانال‌ها

<Callout type="warn" title="پاک کردن همه کانال‌های شنود">
برای پاک کردن تمام کانال‌های شنود:
```
/stp material clear
```

⚠️ **هشدار:** این دستور تمام کانال‌های شنود را حذف می‌کند! ربات دیگر هیچ کانالی را شنود نخواهد کرد.
</Callout>

## تنظیم کانال خروجی

### تنظیم کانال لاگ

<Callout type="info" title="کانال ارسال لاگ">
برای تنظیم کانالی که لاگ‌های خرید در آن ارسال شود:
```
/stp material set_output channel:#material-logs
```

**توضیح:**
- هر زمان کاربری متریال در لاکر بگذارد، پیام لاگ در این کانال ارسال می‌شود
- این قابلیت برای نظارت لحظه‌ای بر خریدها مفید است
- تنظیم این کانال اختیاری است

**نمونه لاگ:**
```
DEP: Amir_Kenzo | Khorde_Ahan:64x | ⁠╭🔴︴𝗟𝗼𝗰𝗸𝗲𝗿𝟭⁠ ✅ | 2025-10-10 20:08:16
```
</Callout>

### حذف کانال خروجی

<Callout type="warn" title="پاک کردن کانال لاگ">
برای پاک کردن کانال خروجی لاگ:
```
/stp material clear_output
```

بعد از اجرای این دستور، دیگر لاگ‌ها در هیچ کانالی ارسال نمی‌شوند (اما شمارش متریال‌ها همچنان ادامه دارد).
</Callout>

## مشاهده گزارش‌ها

### گزارش کلی کاربران

<Callout type="info" title="لیست کاربران و مبالغ">
برای مشاهده لیست کلی کاربرانی که متریال گذاشته‌اند:
```
/mat
```

**نمونه خروجی:**
```
📊 گزارش متریال انبار

User_User 15x | $45,000
Kenzo_Kenzo 8x | $22,000
Miner_Ali: 12x | $38,500

💡 برای نمایش متریال‌های یک IC از /mat-items استفاده کنید.
```

**توضیحات:**
- `15x` = تعداد کل متریال‌ها
- `$45,000` = مبلغ کل قابل پرداخت
</Callout>

### جزئیات مالی

<Callout type="info" title="مشاهده دقیق متریال‌های یک کاربر">
برای مشاهده جزئیات دقیق متریال‌های یک کاربر:
```
/mat-items ic_name:Miner_Ali
```

**نمونه خروجی:**
```
Miner_Ali

⚙️ Ahan: 3x ($9,000)
⚙️ Aluminum: 2x ($6,000)

━━━━━━━━━━━━━━━━━━━━━
```

**نکات:**
- پارامتر `ic_name` لیست کاربرهای که میوه گذاشته اند رو نمایش میده
</Callout>

## ریست کردن متریال‌ها

<Callout type="warn" title="پاک کردن متریال‌های یک کاربر">
برای ریست کردن متریال‌های یک کاربر خاص بعد از پرداخت:
```
/mat-reset ic_name:Miner_Ali confirms:y
```

**⚠️ نکات بسیار مهم:**
1. **تایید اجباری**: برای تایید عملیات حذف، **حتماً** باید `confirms:y` ارسال شود
2. **غیرقابل بازگشت**: این عملیات غیرقابل بازگشت است! قبل از اجرا مطمئن شوید
3. **پرداخت اول**: حتماً قبل از ریست کردن، مبلغ را به کاربر پرداخت کنید
4. **بررسی دقیق**: با دستور `/mat-items` جزئیات را قبل از ریست بررسی کنید

**مثال فرآیند کامل:**
```bash
# 1. مشاهده لیست کل
/mat

# 2. بررسی جزئیات یک کاربر
/mat-items ic_name:Miner_Ali

# 3. پرداخت $45,000 به کاربر (خارج از ربات)

# 4. ریست کردن متریال‌ها
/mat-reset ic_name:Miner_Ali confirms:y
```


</Callout>

## نحوه استفاده (گام به گام)

<Callout type="info" title="راهنمای سریع راه‌اندازی">
**مرحله 1: تنظیم اولیه**
```bash
# افزودن کانال‌های لاکر
/stp material add channel:#locker1
/stp material add channel:#locker2

# تنظیم کانال لاگ (اختیاری)
/stp material set_output channel:#material-logs

# بررسی تنظیمات
/stp material list
```

**مرحله 2: نظارت خودکار**
- ربات به صورت خودکار متریال‌های گذاشته شده را شمارش می‌کند
- در صورت تنظیم کانال خروجی، لاگ‌ها ارسال می‌شوند

**مرحله 3: مشاهده گزارش‌ها**
```bash
# لیست کلی
/mat

# جزئیات یک کاربر
/mat-items ic_name:Miner_Ali
```

**مرحله 4: پرداخت و ریست**
```bash
# بعد از پرداخت مبلغ به کاربر
/mat-reset ic_name:Miner_Ali confirms:y
```
</Callout>

## خلاصه دستورات

<Callout type="info" title="جدول دستورات">
| دستور | کاربرد | نوع |
|-------|---------|------|
| `/stp material add channel:#xxx` | افزودن کانال شنود | تنظیمات |
| `/stp material remove channel:#xxx` | حذف کانال شنود خاص | تنظیمات |
| `/stp material clear` | حذف تمام کانال‌های شنود | تنظیمات |
| `/stp material set_output channel:#xxx` | تنظیم کانال خروجی | تنظیمات |
| `/stp material clear_output` | حذف کانال خروجی | تنظیمات |
| `/stp material list` | نمایش تنظیمات | تنظیمات |
| `/mat` | گزارش کلی کاربران | گزارش |
| `/mat-items ic_name:xxx` | جزئیات متریال‌های کاربر | گزارش |
| `/mat-reset ic_name:xxx confirms:y` | ریست متریال‌های کاربر | مدیریت |
</Callout>
