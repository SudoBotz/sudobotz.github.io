### مثال های استفاده از دستورات
در پایین میتونید نمونه دستورات رو ببینید

### جست‌وجوی پلیر

<Callout type="info" title="جست‌وجوی پلیر">
با استیم هگز:
```
/sunset player identifier:steam:11000010A1B2C3
```
با ایدی عددی:
```
/sunset player identifier:26
```
</Callout>

### اطلاعات کلی سرور

<Callout type="info" title="اطلاعات کلی سرور">
نمایش اطلاعات جزی از سرور
```
/sunset info
```
</Callout>


### گنگ های برتر

<Callout type="info" title="گنگ های برتر">
نمایش لیست تمام گنگ ها براساس لول (فقط گنگ های لول 5 به بالا نمایش داده میشه)
```
/sunset ganglist
```
</Callout>

### تاپ کیلر کپچر

<Callout type="info" title="تاپ کیلر کپچر">
نمایش لیست برترین پلیرهای کپچر
```
/sunset top kill scope:
```
توی بخش scope باید انتخاب کنید که اخرین نفر باشه یا درهفته یا درماه ...

</Callout>

### تاپ پینگ

<Callout type="info" title="تاپ پینگ">
نمایش پلیرها براساس پینگ کمتر
```
/sunset top ping
```
</Callout>

### تاپ ایدی

<Callout type="info" title="تاپ ایدی">
نمایش پلیرها براساس ایدی عددی کمتر
```
/sunset top id
```
</Callout>
