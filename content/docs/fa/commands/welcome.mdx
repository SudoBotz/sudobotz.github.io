## راهنمای دستور

برای دریافت راهنمای سریع دستور می‌توانید از دستور زیر استفاده کنید:

```
/stp wellcome help
```

## Placeholder ها

در پیام‌های خوش‌آمدگویی می‌توانید از متغیرهای زیر استفاده کنید:

<Callout type="info" title="Placeholder های موجود">
این متغیرها به صورت خودکار با اطلاعات عضو جدید جایگزین می‌شوند:

- `{mention}` - منشن کردن عضو جدید (@user)
- `{name}` - نام کاربری عضو
- `{display_name}` - نام نمایشی عضو در سرور
- `{guild}` - نام سرور
- `{member_avatar}` - لینک آواتار عضو
- `{guild_icon}` - لینک آیکون سرور
</Callout>

## فیلدهای قابل تنظیم

<Callout type="info" title="تمام گزینه‌های سفارشی‌سازی">
شما می‌توانید موارد زیر را در پیام خوش‌آمدگویی تنظیم کنید:

**محتوای اصلی**
- **title**: عنوان امبد
- **text**: متن/توضیح امبد
- **color**: رنگ امبد (فرمت HEX مثل `#00FFAA` یا عدد)

**تصاویر**
- **banner**: URL تصویر اصلی (بنر بالای امبد)
- **thumbnail**: URL تصویر کوچک یا استفاده از `{member_avatar}` / `{guild_icon}`

**اطلاعات اضافی**
- **footer_text**: متن فوتر (پایین امبد)
- **footer_icon**: آیکون فوتر
- **author_name**: نام نویسنده (بالای امبد)
- **author_icon**: آیکون نویسنده

**تنظیمات عمومی**
- **channel**: کانال ارسال پیام (مثل `#welcome`)
- **preview**: نمایش پیش‌نمایش پیام (True/False)
</Callout>

## تنظیم پیام خوش‌آمدگویی

### مثال ساده با عنوان و متن

<Callout type="info" title="پیام خوش‌آمدگویی ساده">
تنظیم یک پیام خوش‌آمدگویی با عنوان و متن:
```
/stp wellcome config title:'سلام {display_name}!' text:'خوش اومدی {mention} به {guild}' color:'#00ffaa'
```
این دستور یک پیام خوش‌آمدگویی با عنوان سفارشی، متن خوش‌آمدگویی و رنگ سبز آبی ایجاد می‌کند.
</Callout>

### تنظیم کانال و بنر

<Callout type="info" title="تنظیم کانال و تصویر بنر">
مشخص کردن کانال ارسال و افزودن بنر متحرک:
```
/stp wellcome config channel:#welcome banner:https://.../banner.gif
```
پیام در کانال مشخص شده ارسال می‌شود و یک تصویر GIF به عنوان بنر نمایش داده می‌شود.
</Callout>

### استفاده از آواتار عضو و فوتر

<Callout type="info" title="آواتار و فوتر سفارشی">
افزودن آواتار عضو و پیام فوتر:
```
/stp wellcome config thumbnail:'{member_avatar}' footer_text:'از حضورت خوشحالیم!'
```
آواتار عضو جدید به عنوان تصویر کوچک و یک پیام دوستانه در فوتر نمایش داده می‌شود.
</Callout>

### تنظیم نویسنده و آیکون

<Callout type="info" title="تنظیم بخش نویسنده">
افزودن نام تیم و آیکون سرور:
```
/stp wellcome config author_name:'تیم سرور' author_icon:'{guild_icon}'
```
نام تیم در بالای امبد با آیکون سرور نمایش داده می‌شود.
</Callout>

### مثال کامل

<Callout type="info" title="پیام خوش‌آمدگویی کامل">
تنظیم یک پیام خوش‌آمدگویی با تمام امکانات:
```
/stp wellcome config channel:#︴✋】𝗪𝗲𝗹𝗰𝗼𝗺𝗲 title:'خوش آمدی {display_name}! 🎉' text:'سلام {mention}!\n\nبه سرور **{guild}** خوش اومدی!\nامیدواریم لحظات خوبی رو اینجا داشته باشی.' banner:'https://example.com/welcome-banner.gif' color:'#00ffaa' thumbnail:'{member_avatar}' footer_text:'عضو جدید ما هستی!' footer_icon:'{guild_icon}' author_name:'تیم {guild}' author_icon:'{guild_icon}' preview:True
```
این دستور یک پیام حرفه‌ای و کامل با تمام المان‌ها ایجاد می‌کند و پیش‌نمایش آن را نمایش می‌دهد.
</Callout>

### استفاده از مقادیر عددی برای تنظیمات سریع

<Callout type="info" title="تنظیم سریع با اعداد">
استفاده از اعداد برای تنظیم سریع (1 = فعال):
```
/stp wellcome config channel:#︴✋】𝗪𝗲𝗹𝗰𝗼𝗺𝗲 title:1 text:1 banner:3 color:1 thumbnail:1 footer_text:1 footer_icon:1 author_name:1 author_icon:1 preview:True
```
این روش برای تنظیم سریع و استفاده از مقادیر پیش‌فرض مناسب است.
</Callout>

## پاک کردن تنظیمات

### ریست کردن همه تنظیمات

<Callout type="warn" title="حذف کامل تنظیمات">
پاک کردن تمام تنظیمات خوش‌آمدگویی:
```
/stp wellcome clear all:True
```
⚠️ این دستور تمام تنظیمات را حذف می‌کند!
</Callout>

### حذف انتخابی تنظیمات

<Callout type="info" title="حذف بخش‌های خاص">
شما می‌توانید فقط بخش‌های خاصی را حذف کنید:
```
/stp wellcome clear title:True color:True banner:True thumbnail:True footer:True
```
در این مثال، عنوان، رنگ، بنر، تصویر کوچک و فوتر حذف می‌شوند، اما متن و کانال حفظ می‌شوند.
</Callout>

<Callout type="info" title="ترکیب حذف و حفظ">
مشخص کردن دقیق چه چیزی حذف و چه چیزی حفظ شود:
```
/stp wellcome clear all:True title:True text:False color:True banner:True thumbnail:True footer:True author:False channel:False
```
در این حالت:
- ✅ **حذف می‌شود**: title, color, banner, thumbnail, footer
- ❌ **حفظ می‌شود**: text, author, channel
</Callout>

## نکات مهم

<Callout type="warn" title="نکات استفاده">
1. **فرمت رنگ**: رنگ را حتماً با فرمت HEX وارد کنید (مثل `#00FFAA`)
2. **لینک تصاویر**: لینک‌های تصاویر باید معتبر و در دسترس باشند
3. **Placeholder ها**: از آکولاد `{}` برای Placeholder ها استفاده کنید
4. **پیش‌نمایش**: قبل از تایید نهایی، از `preview:True` استفاده کنید
5. **کانال**: کانال باید با `#` شروع شود (مثل `#welcome`)
</Callout>
